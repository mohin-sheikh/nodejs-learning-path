# Node.js For Beginners

A beginner-friendly Node.js learning path designed for students who already know basic JavaScript and want to start Backend Development.

This repository follows a structured step-by-step approach where each folder represents a learning session. Every session contains detailed explanations, examples, exercises, and practical tasks.

By the end of this course, students will be able to

* Understand Node.js fundamentals
* Work with Node.js modules
* Create HTTP servers
* Build REST APIs
* Perform CRUD operations
* Use Express.js
* Connect with databases
* Understand backend development workflows
* Build real-world backend applications

---

# Prerequisites

Before starting this course, students should be comfortable with

* Variables
* Functions
* Arrays
* Objects
* Loops
* Conditional Statements
* ES6 Concepts

  * let and const
  * Arrow Functions
  * Destructuring
  * Template Literals
  * Modules

If you are not comfortable with the above topics, learn JavaScript basics first and then return to this course.

# How To Use This Repository

Follow the sessions in order.

Do not skip any session.

Recommended learning flow

```text
Read Theory
    ↓
Understand Example
    ↓
Run Example
    ↓
Complete Practice Tasks
    ↓
Move To Next Session
```
